Typography Showcase

Font Families

Georgia, serif - A classic and elegant serif font.

Courier New, monospace - A monospaced font resembling typewriter text.

Verdana, sans-serif - A modern and clean sans-serif font.

Font Sizes

Small Font Size - Perfect for fine print or disclaimers.

Medium Font Size - Ideal for body text in articles and paragraphs.

Large Font Size - Great for headings and important notices.

Font Weights

Normal Font Weight - Standard weight for most text.

Bold Font Weight - Used for emphasis and headings.

Lighter Font Weight - Creates a lighter, airier text feel.

Font Styles

Normal Font Style - The default text appearance.

Italic Font Style - Adds a slanted emphasis to text.

Oblique Font Style - Similar to italic but with a more uniform slant.

Line Heights

Normal Line Height - Standard spacing between lines.

Relaxed Line Height - More space between lines for readability.

Tight Line Height - Less space between lines for compact text.

Text Transforms

Uppercase Text - All letters capitalized for emphasis.

Lowercase Text - All letters in lowercase.

Capitalize Text - Each word starts with an uppercase letter.
