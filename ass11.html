Static Positioning

This is a static positioned element. Static is the default positioning for all elements. Elements are positioned according to the normal document flow.

Relative Positioning

This is a relative positioned element. Relative positioning moves the element relative to its normal position in the document flow. Here it is moved down by 10px and right by 20px.

Sticky Positioning

This is a sticky positioned element. Sticky positioning is a hybrid between relative and fixed positioning. The element is treated as relative until it reaches a specified scroll position, then it is treated as fixed.

Fixed Positioning

This is a fixed positioned element. Fixed positioning removes the element from the normal document flow and positions it relative to the viewport. This element will stay in its position even when the page is scrolled.

Absolute Positioning

This is an absolutely positioned element. Absolute positioning removes the element from the normal document flow and positions it relative to the nearest positioned ancestor (in this case, the .relative-container). This element is positioned 50px from the top and left of its container.
