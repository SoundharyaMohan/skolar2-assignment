CSS Box Model Demonstration

Content Box Model

The content-box model includes only the content area within the specified width and height. Padding, borders, and margins are added to the content area’s size.

This box uses the content-box model.

Border Box Model

The border-box model includes the content area, padding, and border within the specified width and height. Only the margins are added outside the box.

This box uses the border-box model.

Effect of Padding

Padding is the space between the content area and the border. It expands the box’s total size by adding space inside the border.

This box has extra padding.

Effect of Border

Borders are added around the padding and content area. The width of the border affects the total size of the box.

This box has a thicker border.

Effect of Margin

Margins create space outside the border of the box, separating it from other elements.
